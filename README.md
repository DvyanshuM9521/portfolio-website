# Divyanshu Mishra Portfolio

A modern, Vercel-inspired portfolio website showcasing AI systems, enterprise software, cybersecurity platforms, and full-stack engineering projects.

## Live Website

Portfolio: https://portfolio-dvyanshu9521.vercel.app

## Featured Projects


### Self-Healing Smart Microgrid

AI-driven microgrid simulation using Deep Reinforcement Learning (DRL) and Graph Neural Networks (GNN) for fault detection and autonomous recovery.

### Lexora AI

Contract Intelligence Platform that extracts clauses, identifies legal risks, generates summaries, and provides AI-powered contract insights.

### SentinelAI SOC

Cybersecurity Operations Center dashboard for threat monitoring, incident management, and security analytics.
